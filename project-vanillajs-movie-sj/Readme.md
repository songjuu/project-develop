## 1. 팀 과제 소개

팀 레이디퍼스트 Vanilla js를 활용한 영화 검색 사이트입니다.

1페이지는 메인 페이지로, 2페이지는 영화 상세 페이지로 구성했습니다.

메인 페이지에서는 영화를 검색할 수 있고 상세 페이지로 이동할 수 있습니다.

상세 페이지는 해당 영화의 다양한 정보를 확인할 수 있고 영화의 예고편을 볼 수 있습니다. 또한 해당 영화에 대한 리뷰를 작성할 수 있습니다.
